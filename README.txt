# Kalorienrechner

Das ist mein erstes größeres Python-Projekt.

Ich habe den Kalorienrechner gebaut, um die Grundlagen besser zu verstehen und praktisch anzuwenden. Dabei ging es vor allem um Funktionen, Listen, Dictionaries, Schleifen und das Speichern von Daten mit JSON. Zusätzlich habe ich noch eine einfache Desktop-Version mit Tkinter ausprobiert.

## Funktionen

### Konsolen-Version (Hauptversion)

- Produkte hinzufügen
- Produkte speichern und laden (JSON)
- Mahlzeiten hinzufügen (mit Grammangaben)
- Mahlzeiten speichern und laden
- Produktliste anzeigen
- Mahlzeitenliste anzeigen
- Gesamtwerte berechnen
- Tageswerte für ein bestimmtes Datum berechnen

### Desktop-Version (Zusatz)

- Produkte eingeben (Name + Nährwerte)
- Eingaben prüfen
- Produkte anzeigen
- Gesamtwerte berechnen

Die Desktop-Version ist deutlich einfacher und dient nur als kleine Erweiterung.

## Dateien

- kalorienrechner_konsole.py → Hauptprogramm
- kalorienrechner_desktop.py → einfache Desktop-Version

## Starten

Konsolen-Version:
python kalorienrechner_konsole.py

Desktop-Version:
python kalorienrechner_desktop.py

## Hinweis

Die Konsolen-Version ist die vollständige Version vom Projekt.  
Die Desktop-Version ist nur ein einfacher Zusatz zum Ausprobieren von Tkinter.
